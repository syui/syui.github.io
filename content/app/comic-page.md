`comic.page`は、vue.jsで書かれたシンプルな画像ページを作ります。

[Download](https://github.com/syui/comic.page)

demo : https://syui.github.io/comic.page

```sh
$ git clone https://github.com/syui/comic.page
$ cd comic.page
$ yarn install
$ yarn dev
$ yarn build
```
