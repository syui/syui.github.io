### bluesky

`bluesky`では[rate limit](https://atproto.com/blog/rate-limits-pds-v3)の変更がありました。

この変更によってbotなどが制限を受けやすくなります。

ここでは、実際にblueskyの負荷を減らし、botが制限されることを回避する一例を紹介します。

具体的には、`createSession`を減らし、代わりに`refreshSession`を使用するようにします。

createSession
- 30/5 min
- 300/day

#### refresh token

今までは`createSession`で都度、tokenを取得し、新しいsessionを作成してきました。

`create`は新しいパスワードを作るイメージ、`refresh`はパスワードを更新するイメージです。これらには有効期限が付いています。有効期限付きのパスワードをtokenと呼びます。

`createSession`でapiを呼び出す都度、新しいtokenを発行すれば、たしかに確実ですが、有効なtokenがいくつもできてしまいます。これは、危険でもあり、無駄でもあります。ですから、`refreshSession`で一つのパスワードを更新して使い回す方法が用意されています。`refresh-token`とも呼ばれます。

基本的には、`com.atproto.server.refreshSession`にtokenとして`refreshJwt`を指定します。

```sh
handle=xxx.bsky.social
pass=xxx
curl -sL -X POST -H "Content-Type: application/json" \
         -d "{\"identifier\":\"$handle\",\"password\":\"$pass\"}" \
         https://bsky.social/xrpc/com.atproto.server.createSession \
         | jq -r .refreshJwt

refresh=`!!`

curl -sL -X POST -H "Content-Type: application/json" -H "Authorization: Bearer $refresh" https://bsky.social/xrpc/com.atproto.server.refreshSession
```

例えば、これを`rust`で書いてみます。

```sh
├── Cargo.toml
└── src
    ├── main.rs
    └── refresh.rs
```

```toml:Cargo.toml
[package]
name = "ai"
version = "0.1.0"
edition = "2021"

[dependencies]
seahorse = "*"
reqwest = { version = "*", features = ["blocking", "json"] }
tokio = { version = "1", features = ["full"] }
```

```rust:src/refresh.rs
extern crate reqwest;

pub async fn post_request() -> String {

    let refresh = "xxxxxxxxxxxxxxxxxxxxxxxxxxx";

    let url = "https://bsky.social/xrpc/com.atproto.server.refreshSession";

    let client = reqwest::Client::new();
    let res = client
        .post(url)
        .header("Authorization", "Bearer ".to_owned() + &refresh)
        .send()
        .await
        .unwrap()
        .text()
        .await
        .unwrap();

    return res
}
```

```rust:src/main.rs
pub mod refresh;

use seahorse::{App, Command, Context};
use std::env;

fn main() {
    let args: Vec<String> = env::args().collect();
    let app = App::new(env!("CARGO_PKG_NAME"))
        .command(
            Command::new("refresh")
            .alias("r")
            .action(c_refresh),
            )
        ;
    app.run(args);
}

fn refresh() {
    let h = async {
        let str = refresh::post_request().await;
        println!("{}",str);
    };
    let res = tokio::runtime::Runtime::new().unwrap().block_on(h);
    return res
}

fn c_refresh(_c: &Context) {
    refresh();
}
```
