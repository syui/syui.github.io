### archlinux

まず、archlinuxのisoをdownloadします。ここでは`jaist`, `tsukuba`のftpを紹介します。どちらも日本にあるserverなので速度的には有利だと思われます。

http://ftp.jaist.ac.jp/pub/Linux/ArchLinux/iso/latest/

http://ftp.tsukuba.wide.ad.jp/Linux/archlinux/iso/latest/

archlinuxの公式ページは`archlinux.org`と`archlinux.jp`があります。

en : [https://archlinux.org](https://archlinux.org)

ja : [https://archlinux.jp](https://archlinux.jp)

`.org`が本家、`.jp`が日本コミュニティです。

`iso`をdownloadできたら、それをusbに置きます。usbは2Gもあれば十分でしょう。

```sh
$ sudo dd if=./archlinux.iso of=/dev/sdx bs=4M
```

これは色々なやり方があります。

```sh
# cat path/to/archlinux-version-x86_64.iso > /dev/disk/by-id/usb-My_flash_drive
# cp path/to/archlinux-version-x86_64.iso /dev/disk/by-id/usb-My_flash_drive
```

まずusbがどこに接続されているかを調べます。そこを指定して、`dd`や`cat`のコマンドを実行します。現在のosがインストールされているdiskを指定しないよう注意してください。この例では`sda`を指定してはいけません。`sdb`を指定します。

```sh
$ sudo fdisk -l
sda ... 100G
sdb ... 2G

$ sudo dd if=~/Download/archlinux-version-x86_64.iso of=/dev/sdb bs=4M
```

なお、`mac`では`diskutil`が必要です。

出来上がったusbをパソコンに挿して、usb-boot(usbからimgを起動)して、archlinuxをインストールします。

```sh
$ archinstall
```

archlinuxは、非常にマニアックなosですが、最先端のエンジニアに人気があります。

最近でいうと、mac-m1にインストールできる[asahi linux](https://asahilinux.org/about)というものがあって、archlinuxが採用されています。

また、[steam deck](https://www.steamdeck.com/ja/tech)というゲーム機が販売されていますが、その`steam os`は、archlinuxをベースにしています。
