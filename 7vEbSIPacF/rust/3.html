bluesky

bluesky

blueskyではrate limitの変更がありました。

この変更によってbotなどが制限を受けやすくなります。

ここでは、実際にblueskyの負荷を減らし、botが制限されることを回避する一例を紹介します。

具体的には、createSessionを減らし、代わりにrefreshSessionを使用するようにします。

createSession

  • 30/5 min
  • 300/day

refresh token

今まではcreateSessionで都度、tokenを取得し、新しいsessionを作成してきました。

createは新しいパスワードを作るイメージ、refreshはパスワードを更新するイメージです。これらには有効期限が付いています。有効期限付きのパスワードをtokenと呼びます。

createSessionでapiを呼び出す都度、新しいtokenを発行すれば、たしかに確実ですが、有効なtokenがいくつもできてしまいます。これは、危険でもあり、無駄でもあります。ですから、refreshSessionで一つのtokenを更新して使い回す方法が用意されています。refresh-tokenとも呼ばれます。

基本的には、com.atproto.server.refreshSessionにtokenとしてrefreshJwtを指定します。

handle=xxx.bsky.social
pass=xxx
curl -sL -X POST -H "Content-Type: application/json" \
         -d "{\"identifier\":\"$handle\",\"password\":\"$pass\"}" \
         https://bsky.social/xrpc/com.atproto.server.createSession \
         | jq -r .refreshJwt

refresh=`!!`

curl -sL -X POST -H "Content-Type: application/json" -H "Authorization: Bearer $refresh" https://bsky.social/xrpc/com.atproto.server.refreshSession

例えば、これをrustで書いてみます。

Cargo.toml

[package]
name = "ai"
version = "0.1.0"
edition = "2021"

[dependencies]
seahorse = "*"
reqwest = { version = "*", features = ["blocking", "json"] }
tokio = { version = "1", features = ["full"] }

src/refresh.rs

extern crate reqwest;

pub async fn post_request() -> String {

    let refresh = "xxxxxxxxxxxxxxxxxxxxxxxxxxx";

    let url = "https://bsky.social/xrpc/com.atproto.server.refreshSession";

    let client = reqwest::Client::new();
    let res = client
        .post(url)
        .header("Authorization", "Bearer ".to_owned() + &refresh)
        .send()
        .await
        .unwrap()
        .text()
        .await
        .unwrap();

    return res
}

src/main.rs

pub mod refresh;

use seahorse::{App, Command, Context};
use std::env;

fn main() {
    let args: Vec<String> = env::args().collect();
    let app = App::new(env!("CARGO_PKG_NAME"))
        .command(
            Command::new("refresh")
            .alias("r")
            .action(c_refresh),
            )
        ;
    app.run(args);
}

fn refresh() {
    let h = async {
        let str = refresh::post_request().await;
        println!("{}",str);
    };
    let res = tokio::runtime::Runtime::new().unwrap().block_on(h);
    return res
}

fn c_refresh(_c: &Context) {
    refresh();
}
